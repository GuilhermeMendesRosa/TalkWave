## 🌊 TalkWave 🌊

O TalkWave é um aplicativo de mensagens desenvolvido em Java que permite comunicação em tempo real por meio de sockets em uma rede local.

Este projeto foi desenvolvido como parte do curso de TADS da UDESC e foi inspirado no projeto [MessageApp](https://github.com/GuilhermeMendesRosa/MessageApp).
## Tecnologias Utilizadas
- ☕ Java
- 🌐 Sockets
- 🧵 Threads
